# AutoTheme (自适应主题)

让我的世界的任务栏自适应系统主题！
- 黑/白（Windows10-11）（X64/X86）
+        10 1809+
-        11 任意版本

---

## 模组功能？/ 不同点？

- [Windows] 自动切换黑白主题
+ （使用DLL来获取系统黑白主题并做到实时切换）
+ （支持X64/X86架构）
+ 最快速动态主题检测/应用
- [More] ...

---

## 本模组已知问题

- [Minecraft] 无响应时窗口返回浅色

---

## 本模组需要的依赖？

- Minecraft [More...] 1.20-1.21.9 [可直接最新版本]
+ Java21+
- Fabric Loader[**最新版本**]

---

## 如何安装本模组？

1. 安装FabricLoader最新版本
2. 将模组放入整合包目录下`mods\`文件夹内
