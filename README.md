# AutoTheme (自适应主题)

让我的世界的任务栏自适应系统主题！
- 黑/白（Windows10-11）（X64）
+        10 1809+
-        11 任意版本

---

## 模组功能？/ 不同点？

- [Windows]自动切换黑白主题
+ （使用DLL来获取系统黑白主题并做到实时切换）
- [More]...

---

## 本模组已知问题

- [Minecraft] 无响应时窗口返回浅色
- [本模组] 缺少X86支持（计划中）

---

## 本模组需要的依赖？

- Minecraft 1.21-1.21.1
+ Java21+
- Fabric最新

---

## 如何安装本模组？

1. 安装FabricLoader最新版本
2. 将模组放入整合包目录下`mods\`文件夹内
